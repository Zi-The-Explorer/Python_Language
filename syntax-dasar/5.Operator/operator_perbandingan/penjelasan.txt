Operator perbandingan (comparison operators) digunakan untuk membandingkan suatu nilai dari masing-masing operan.

Operator                        Contoh                                                              Penjelasan

==                             x == y                                                            Mengembalikan nilai true jika x sama dengan y
!=                            x != y                                                            Mengembalikan nilai true jika x tidak sama dengan y
<>                            x <> y                                                            Mengembalikan nilai true jika x tidak sama dengan y
>                             x > y                                                             Mengembalikan nilai true jika x lebih besar dari y
<                             x < y                                                             Mengembalikan nilai true jika x lebih kecil dari y
>=                            x >= y                                                            Mengembalikan nilai true jika x lebih besar atau sama dengan y
<=                            x <= y                                                            Mengembalikan nilai true jika x lebih kecil atau sama dengan y
