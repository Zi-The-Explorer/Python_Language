Di Python untuk mencetak cukup gunakan fungsi print() ,
dimana sesuatu yang akan dicetak harus diletakkan diantara kurung buka dan kurung tutup, 
bahkan di Python versi 2.x Anda tidak harus menggunakan tanda kurung kurawal, cukup pisahkan dengan spasi.
Jika ingin mencetak tipe data String langsung, Anda harus memasukanya ke dalam tanda kutip terlebih dahulu.
